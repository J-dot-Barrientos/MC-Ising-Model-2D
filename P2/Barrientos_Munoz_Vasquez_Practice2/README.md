Proyecto computacional en Modelización Molecular (Matteo Palassini)

El repositorio contiene:
- El código optimizado en el directorio `Optimized_Code`
- Simulaciones de 1e3 MCS en `1000steps`
- Simulaciones de 1e6 MCS en `Production`
- Ficheros de salida y python scripts para visualización en `Resultados`
- Código de Binning para análisis estadístico en `Analysis`
